# Descripción general

## ¿Qué es Clearance?

Clearance es un **portal de calidad open source** para datos de OpenStreetMap. Se interpone entre la base de datos mundial de OpenStreetMap y sus aplicaciones para filtrar y validar los cambios antes de que lleguen a sus sistemas.

## Por qué existe Clearance

OpenStreetMap es una base de datos colaborativa abierta. Cualquiera puede modificar los datos, lo cual es su mayor fortaleza — pero también un desafío para las organizaciones que dependen de ella.

Problemas comunes:

- **Ediciones accidentales** — Un contribuidor bien intencionado mueve un punto de interés por error
- **Vandalismo** — Modificaciones maliciosas que corrompen los datos
- **Inconsistencias** — Tags incorrectos o geometrías inválidas
- **Eliminaciones no deseadas** — Objetos eliminados sin razón válida

Clearance resuelve estos problemas estableciendo un control de calidad entre OpenStreetMap y sus aplicaciones.

## Principio de funcionamiento

```
OpenStreetMap → Replicación → Clearance → Sus aplicaciones
                                 ↓
                          Cambios conformes → Pasan automáticamente
                          Cambios sospechosos → Revisión humana
```

1. **Replicación** — Clearance se conecta al flujo de replicación OSM
2. **Análisis** — Cada cambio se evalúa según las reglas de validación
3. **Filtrado** — Los cambios conformes se integran automáticamente
4. **Revisión** — Los cambios sospechosos se presentan para validación humana
