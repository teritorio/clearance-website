# Roadmap

Clearance is under active development. Here are the main directions for upcoming versions.

## NLNet funding

Clearance development is supported by **NLNet** as part of the NGI (Next Generation Internet) program. This funding accelerates the development of key features.

## External data sources

**Goal**: Enrich Clearance analysis by integrating external data sources for comparison and validation.

- Comparison with reference databases (cadastre, address databases, etc.)
- Regression detection against official data
- Cross-validation with other open sources

## Reputation system

**Goal**: Refine filtering by taking into account contributor history and reliability.

- Trust score per contributor based on history
- Adaptive validation thresholds based on reputation
- Progressive learning from validation decisions

## Interface improvements

- Dashboard with validation statistics
- Enhanced map visualization of LoCha clusters
- Configurable notifications and alerts

## Contributing

Clearance is open source. Contributions are welcome on the [GitHub repository](https://github.com/teritorio/clearance).

- Report bugs or suggest improvements via issues
- Submit a pull request
- Join the discussions
